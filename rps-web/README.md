Rock Paper Scissors — browser app

This is a minimal single-page Rock–Paper–Scissors game implemented with vanilla HTML/CSS/JS.

Run locally (from the repository root):

```bash
python3 -m http.server 8000 --directory repositories/rps-web
```

Then open http://localhost:8000/ in your browser.

Controls:
- Click Rock / Paper / Scissors to play a round.
- First to 2 wins the match.
- Click "Play again" to reset.
