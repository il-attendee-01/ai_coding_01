# Fibonacci and Factorial Calculator

This project contains two functions to calculate the Fibonacci sequence and the factorial of a number.

## Functions

### fibonacci(n)
- **Description**: Generates the Fibonacci sequence up to the nth term.
- **Parameters**:
  - `n` (int): The number of terms in the Fibonacci sequence to generate.
- **Returns**:
  - list: A list containing the Fibonacci sequence up to the nth term.

### factorial(n)
- **Description**: Computes the factorial of a non-negative integer.
- **Parameters**:
  - `n` (int): The non-negative integer to compute the factorial for.
- **Returns**:
  - int: The factorial of the given integer.
- **Raises**:
  - ValueError: If the input is negative.
